# Книжарница (Book Library) - Контейнеризация на проект с Docker Compose

## Описание
Това е уеб приложение за управление на книги, направено с PHP и MySQL. Проектът е контейнеризиран с Docker и Docker Compose, така че може да се стартира лесно на всяка система без допълнителни настройки.

Чрез приложението могат да се:
- добавят книги
- редактират книги
- изтриват книги
- търсят книги

## Docker архитектура

Проектът се състои от два основни контейнера:

### Web (PHP + Apache)
Този контейнер изпълнява PHP приложението и отговаря за уеб интерфейса.

### Database (MySQL)
Тук се съхраняват всички данни за книгите.

---

## Връзка между контейнерите

PHP приложението се свързва към базата данни чрез Docker мрежата, като се използва името на услугата:
host: db
user: root
password: root
database: books_db

Docker Compose автоматично осигурява комуникация между контейнерите.

---

## Стартиране на проекта

### 1. Стартиране на контейнерите
```bash
docker compose up -d --build
```

### 2. Отваряне в браузър
http://localhost:8080

### 3. Спиране на контейнерите
```bash
docker compose down
```

---

## Как работи проектът

Проектът е разделен на два контейнера (web и database), които работят заедно чрез Docker Compose. Това позволява приложението да се стартира с една команда и да работи независимо от операционната система.

## Заключение

Този проект показва как едно уеб приложение може да бъде контейнеризирано и разделено на отделни услуги с Docker. Това улеснява стартиране, преносимост и управление на приложението.
